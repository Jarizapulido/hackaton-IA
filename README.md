# TrainerSurvivor

Este proyecto consta esencialmente de una aplicacion web con la la función principal de crear rutinas para cualquier tipo de deporte, además de ser bastante personalizable
al poder añadir ejercicios nuevos y agrupados por musculos, la otra función principal seria poder activar el modo survivor y poder jugar a un juego roguelike como 
el VampireSurvivor durante los descansos y que con cada ejercicio acabado se podrá elegir una habilidad con la que mejorar al personaje hasta el final de la rutina que habrá que enfrentarse a un boss.

##  (Funcionalidades extra posibles)

- Ponerle niveles de dificultad a los ejercicios y conforme más dificiles sean mejores son las estadisticas de las habilidades dadas
- Que la dificultad del juego aumente progresivamente conforme dura la rutina
